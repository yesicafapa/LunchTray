# Lunch Tray App

Project ini dibuat sebagai bagian dari tugas pada codelab berikut:  
https://developer.android.com/codelabs/basic-android-kotlin-compose-practice-navigation?hl=id#0  

Aplikasi *Lunch Tray* adalah aplikasi pemesanan makanan sederhana yang dibangun menggunakan **Jetpack Compose**, **Material 3**, dan **Navigation Compose**.  
Melalui project ini, saya mempraktikkan beberapa konsep penting seperti:
- Navigasi antar layar (multi-screen navigation)
- Pengelolaan state pada Compose
- Passing data antar screen
- Struktur UI berbasis deklaratif
